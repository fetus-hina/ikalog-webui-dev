IkaLog WebUI
============

開発中専用レポジトリ

[IkaLogのレポジトリはこちらです](https://github.com/hasegaw/IkaLog)


Requirement (developer only)
----------------------------

- node.js （新しいもの）


How to develop
--------------

1. `$ npm install`
2. modify something
3. `$ npm run build` or `$ npm run watch`
